# プロダクト開発: 10年後のZEN Study

## コンセプト
- 現実世界に介入してくるAIとの新しい学び
- 物理エンジンだけでなく、化学エンジンを活用した実践的な学び
- VR・ARを用いた、体験型の学び

## 担当別ワーク
UI：いっちー
概要/教材：The Greatest Technician That's Ever Lived
機能:nikkun


### 概要・ターゲット・ビジネスモデル
- 概要:現在あるVRやAIだけでなく、ARや物理エンジン、化学エンジン等の新しい技術を用いて全日制と遜色のない授業体験を受けることを可能にする
- ターゲット: 身体やメンタルが不安定である人間かつ全日制に近い授業体験を受けたい人

### 画面の構成・UI
**メインページ**
- ヘッダー
  - ロゴ
  - ホーム
  - マイコース
  - フォーラム
  - AIチャット
  - ライブ授業
  - 通知
  - アイコン
    - ユーザー情報
- サイドバー
  - ホーム
  - 必修授業
  - 課外授業
- メイン
  - あなたの学習履歴
  - 最新の授業
  - おすすめコース
- フッター
  - 各種規約等
  - コピーライト

**教材ページ**
- サイドバー
  - ヘッダー
    - ロゴ
    - ホーム
    - マイコース
    - フォーラム
    - AIチャット
    - ライブ授業
    - 通知
    - アイコン
      - ユーザー情報
  - 教材タイトル・説明
  - コース
- メイン
  - 動画
  - 教材資料
  - 物理・科学エンジンで実験する

**各画面の配置**
- ヘッダー
  - 横並びで配置
- ロゴ
  - メインページのロゴは文字入り
  - 教材ページのロゴはロゴのみ
- 未ログイン時は**アイコン**を以下に変更
  - 会員登録
  - ログイン
- 教材ページの**物理・科学エンジンで実験する**
  - 物理や科学などの科目でのみ表示
  - クリックで物理・科学エンジン実験画面に遷移


### 機能一覧
- AR / VRでの実践機能(化学実験など)
- 動画内のわからないところをAIに質問
- 学習状況から得意と苦手を算出
- 物理エンジンだけでなく、化学エンジンを導入
- カメラ越しでAIに質問できる(科学の現象名や国語の読解など)

- 既存の授業に機能を追加

# プロダクト開発: 10年後のZEN Study

## コンセプト
- 現実世界に介入してくるAIとの新しい学び
- 物理エンジンだけでなく、化学エンジンを活用した実践的な学び
- VR・AR、処理エンジンを用いた、体験型の学び

## 担当別ワーク
UI：いっちー
概要/教材：The Greatest Technician That's Ever Lived
機能:nikkun


### 概要・ターゲット・ビジネスモデル
- 概要:現在あるVRやAIだけでなく、ARや物理エンジン、化学エンジン等の新しい技術を用いて全日制と遜色のない授業体験を受けることを可能にする
- ※科学エンジンという用語はないが、将来的に物理エンジンともう一つのエンジンとして出てくると予想する。ゼルダの伝説からの用語｜https://www.4gamer.net/games/341/G034168/20170303060/

  - クリックで物理・科学エンジン実験画面に遷移


### 機能一覧
- AR / VRでの実践機能(化学実験など)、加えて科学エンジンと組み合わせたリアリティある化学実験や実技。
- バーチャル交流学園や、リアルタイムでバーチャル上で授業を受けられる学習教室があるかも
  └AIの技術も進歩していると思うのである程度現実味あり
- 動画内のわからないところをAIに質問
- 学習状況から得意と苦手を算出
- カメラ越しでAIに質問できる(科学の現象名や国語の読解など)
